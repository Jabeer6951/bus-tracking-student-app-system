#  Bus Tracking Student App

An Android application developed for students to track college buses in real time using GPS and Firebase.

---

##  Features

- Live Bus Tracking
- Google Maps Integration
- Student Login & Registration
- View Bus Route Details
- Driver Information
- Bus Miss Detection System
- Upcoming Nearby Buses
- SOS Emergency Feature
- Firebase Realtime Database
- Firebase Authentication

---

##  Technologies Used

- Android Studio
- Kotlin
- XML
- Firebase Authentication
- Firebase Realtime Database
- Google Maps API

---

##  Modules

### Student Dashboard
- View assigned bus
- View route details
- View driver details
- Track bus location live

### SOS System
- Emergency call
- Emergency SMS alerts

### Bus Miss Detection
- Detects if bus crossed student stop
- Shows nearby upcoming buses
